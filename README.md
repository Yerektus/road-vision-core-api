# RoadVision Core API

Данный проект представляет собой REST API для обработки и мониторинга состояния дорожной инфраструктуры с использованием машинного обучения (ML). Решение предназначено для автоматического выявления дефектов дорожного покрытия и предоставления результатов.

## Использованный стек

- Python 3.14
- Fast API
- SQLAlchemy
- Alembic
- FastAPIUsers
- Docker

## Установка

1. Клонирование репозитории
```
git clone https://github.com/Feesder/road-vision-core-api.git
cd road-vision-core-api
```

2. Настройка конфигурации
Создайте конфигурационный файл на основе шаблона и укажите параметры:
```
cp ./.env.example ./.env
```

3. Запуск приложения
```
docker compose up
```
